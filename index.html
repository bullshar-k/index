DNS: Adguard Home 28 Jun 2026

Initial setup was very easy, 10/10, bravo to the devs!

My biggest obstacle was using the Arch Linux /extra Mullvad Client specifically in getting it to work with the local DNS. See /etc/resolv.conf would have 127.0.0.53 which is fine because its pointing back to a local adress for DNS. Mullvad writes this rule. I had to enable LAN sharing, disable lockdown mode, and add custom dns for my local address of the Adguard VM. If I edited resolv.conf to point to 192.168.1.50 (Adguard Homes LAN address) it would send queries out, upstream, cache, then resolve it to the client but it wouldn't end up getting the query back. It's because Mullvad doesn't change your NIC's DNS address when you put a custom DNS. It changes what it needs to but I had to use resolvctl to change the DNS of eno1.

Another incovenient thing is if you want whole network VPN usage you have to manually configure every exit node with a wireguard config. Which is fine for normal server uses, security, privacy of the server etc. However, for normal VPN usage using the client is much easier and more appropriate. Because of exit node outages and some things being randomly banned on some exit nodes, switching often is common and manually configuring & switching each time in my firewal is totally not appropriate for my use.

Firewall: OPNsense 19 Jun 2026

Skipping all the basic stuff in the docs. As a result of the great work of these devs I can't say much about the installation process. It was extremely easy and unremarkable. However, I did have it crash one time. I'd recommend when making the OPNsense VM to turn off memory banding which I beleive was the cause. I also gave it 8/16GB and changed it to 12/16GB with memory banding off which is on by default. It's very nice seeing so much blocked traffic that was not needed. Roughly 70-75% of traffic at any given time. At the same time I replaced my ISPs router and modem. I did have to call them to get the MAC of my old modem wipe & the new one inserted which was annoying, but it didn't take long.
